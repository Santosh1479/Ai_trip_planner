# AI Travel Planner

## Overview
AI Travel Planner is an AI-powered travel planning application that offers personalized recommendations and smooth trip management. The application utilizes advanced technologies to create tailored itineraries based on user input.

## Project Structure
The project is divided into two main parts: the backend and the frontend.

### Backend
The backend is built using Node.js and Express. It handles API requests, manages the database, and implements the AI logic for trip planning.

- **`backend/`**: Contains all backend-related files.
  - **`package.json`**: Configuration file for npm, listing dependencies and scripts.
  - **`.env.example`**: Template for environment variables.
  - **`src/`**: Source code for the backend application.
    - **`index.js`**: Entry point for the backend server.
    - **`config/`**: Contains database configuration.
    - **`models/`**: Defines the Trip model.
    - **`routes/`**: Defines routes for trip planning.
    - **`controllers/`**: Contains request handling logic.
    - **`services/`**: Implements AI planning logic.

### Frontend
The frontend is built using React and provides a user-friendly interface for trip planning.

- **`frontend/`**: Contains all frontend-related files.
  - **`package.json`**: Configuration file for npm, listing dependencies and scripts.
  - **`index.html`**: Main HTML file for the frontend application.
  - **`postcss.config.js`**: Configuration for PostCSS.
  - **`tailwind.config.js`**: Configuration for Tailwind CSS.
  - **`src/`**: Source code for the frontend application.
    - **`main.jsx`**: Entry point for the React application.
    - **`App.jsx`**: Main App component.
    - **`api.js`**: Functions for making API calls to the backend.
    - **`components/`**: Contains reusable components for the application.
      - **`Navbar.jsx`**: Navigation component.
      - **`TripForm.jsx`**: Form for user input.
      - **`Itinerary.jsx`**: Displays the planned itinerary.
      - **`DayCard.jsx`**: Represents a single day in the itinerary.
      - **`LoadingDots.jsx`**: Indicates loading states.
    - **`styles.css`**: Main CSS styles for the frontend application.

## Setup Instructions

### Backend
1. Navigate to the `backend` directory.
2. Install dependencies:
   ```
   npm install
   ```
3. Create a `.env` file based on `.env.example` and configure your environment variables.
4. Start the server:
   ```
   npm start
   ```

### Frontend
1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm start
   ```

## Usage
1. Open the frontend application in your browser.
2. Fill out the trip details in the TripForm component.
3. Submit the form to receive a personalized itinerary generated by the AI.
